Design a complete modern web application UI for a Blockchain-Enabled Civic FinTech Accountability Platform.

The application helps citizens report civic issues and transparently track government fund usage using blockchain technology.

Create multiple screens with consistent design system and navigation.

--------------------------------
SCREEN 1 — HOME DASHBOARD
--------------------------------
Include:
- Left sidebar navigation (Home, Report Issue, Admin Dashboard, Public Transparency)
- Top navbar with city name and profile icon
- Statistics cards:
  • Total Issues Reported
  • Funds Allocated
  • Funds Released
  • Average Resolution Time
- Recent Issues section displayed as cards
- Clean dashboard layout

--------------------------------
SCREEN 2 — REPORT ISSUE PAGE
--------------------------------
Include:
- Page title: "Report Civic Issue"
- Form fields:
  • Issue Title input
  • Category dropdown (Pothole, Garbage, Streetlight, Water Issue)
  • Description textarea
  • Upload Image button
  • Location input field
  • Submit button
- Modern form layout with rounded inputs

--------------------------------
SCREEN 3 — ADMIN DASHBOARD
--------------------------------
Include:
- Table listing reported issues
Columns:
  • Issue ID
  • Category
  • Location
  • Status
  • Allocated Fund
  • Actions
- Fund allocation panel/modal
- Status update dropdown:
  Open / In Progress / Resolved
- Professional admin dashboard style

--------------------------------
SCREEN 4 — PUBLIC TRANSPARENCY DASHBOARD
--------------------------------
Include:
- Fund Utilization chart
- Issue Resolution Time chart
- Ward performance table
- Map placeholder showing issue locations
- Analytics dashboard layout

--------------------------------
DESIGN STYLE
--------------------------------
- Modern government-tech interface
- Clean minimal design
- Blue and white color theme
- Soft shadows and rounded cards
- Dashboard-style UI
- Consistent spacing and typography
- Professional smart city appearance
- Responsive web layout (desktop first)

--------------------------------
EXTRA REQUIREMENTS
--------------------------------
- Create reusable components (buttons, cards, tables, navbar)
- Maintain consistent design system
- Use modern sans-serif font (Inter or Poppins)
- Ensure clean alignment and hierarchy
